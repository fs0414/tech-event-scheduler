# Tech Event Scheduler サービス紹介

## サービス概要

Tech Event Schedulerは、技術イベントの運営管理をサポートするWebアプリケーションです。勉強会やLT会などの技術イベントで必要な機能を提供します。

## 主な機能

### イベント管理
- イベントの作成・編集・削除
- イベント一覧表示と検索機能
- 外部イベントページへのリンク設定
- 参加者数の記録

### 運営者管理
- 複数の運営者を設定可能
- 管理者とメンバーの2つの権限レベル
  - 管理者（role: 10）: イベントの全機能を利用可能
  - メンバー（role: 20）: 限定的な権限で運営サポート
- チームでのイベント運営に対応

### スピーカー管理
- 登壇者の登録と管理
- 発表資料（Article）の関連付け
- スピーカーごとの発表履歴管理

### タイマー機能
- 発表時間の設定（分単位）
- 複数タイマーのシーケンス管理
- LT会や勉強会の時間管理をサポート

## 使い方

### 1. アカウント作成・ログイン
Supabase Authを使用した認証システムで、メールアドレスでアカウントを作成します。

### 2. イベント作成
- イベントタイトルを入力
- 必要に応じて外部イベントページのURLを設定
- 参加予定者数を入力

### 3. 運営チーム構成
- 他のユーザーを運営者として追加
- 役割（管理者/メンバー）を設定

### 4. スピーカー登録
- 登壇者情報を登録
- 発表資料のリンクを追加

### 5. タイムスケジュール設定
- 各発表の持ち時間を設定
- 発表順序を管理

## こんな場面で便利

### 社内勉強会の運営
- 定期的な技術勉強会の情報を一元管理
- 過去の発表資料をアーカイブとして保存
- 誰がいつ何を発表したか履歴を確認

### LT会の時間管理
- 各発表者の持ち時間を事前設定
- タイマー機能で円滑な進行
- 複数の短い発表を効率的に管理

### コミュニティイベントの運営
- 複数の運営メンバーで役割分担
- スピーカー情報を整理して参加者に共有
- イベント実績を蓄積

## 技術構成

- **フロントエンド**: Next.js 15 (App Router)、TypeScript、React 19
- **スタイリング**: Tailwind CSS、shadcn/ui
- **データベース**: PostgreSQL (Supabase)
- **ORM**: Prisma
- **認証**: Supabase Auth
- **ホスティング**: Vercel

## データモデル

- **User**: ユーザー情報（認証連携）
- **Event**: イベント基本情報
- **Owner**: イベント運営者（ユーザーとイベントの関連）
- **Speaker**: 登壇者情報（ユーザーとイベントの関連）
- **Article**: 発表資料情報
- **Timer**: タイマー設定

## 今後の展開可能性

現在の機能をベースに、以下のような拡張が考えられます：
- 参加者の出欠管理
- イベント後のフィードバック収集
- 統計情報の可視化
- カレンダー連携
- 通知機能

Tech Event Schedulerは、技術イベントの運営に必要な基本機能を提供し、運営者の負担を軽減するツールです。
